What are these plugins?!

A plugin is basically a software that is used to make music. These plugins that you see before you have alot of knobs and various parameters you can change. These knobs and parameters can be changed to make the sounds that you desire. Harmor,the plugin at the very left, can be used to make growls,pluck sounds,vox voices,and many more sounds. Massive, the plugin at the very right, is what skrillex used to make scary monsters and nice sprites! Cool right.
